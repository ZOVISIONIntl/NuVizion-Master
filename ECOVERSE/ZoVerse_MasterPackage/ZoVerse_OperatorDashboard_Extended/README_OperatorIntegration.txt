ZoVerse Operator Dashboard Integration

This extension layer integrates the following automations:
- Notarization (AutoNotarize_Documents.py)
- IP Logging (IP_Log_Registrar.py)
- Contract Injection (Contract_Injector_Agent.js)
- Dawn & Dust Task Sync (LiveSync_DawnDust.json)

Usage:
- Load these into your Operator Dashboard backend
- Assign trigger buttons in the HTML or CLI interface
- Connect each function to input/output file systems for live results
